Section 04

Current Pain Points

What we need to solve in this engagement.

1. Manual Path Classification

Problem: Every opportunity must be manually categorized by path. No automation.

Impact: Time-consuming, inconsistent classification, missed opportunities.

Desired: Auto-classify based on notice type, agency, set-aside, and keywords.

2. No Weighted Value Calculation

Problem: Pipeline shows total value only. No automated weighting by win rate.

Impact: Can't accurately forecast revenue or prioritize effort.

Desired: Auto-calculate weighted value = Deal Value Ã— Path Win Rate.

3. Deadline Management is Manual

Problem: Response deadlines tracked in spreadsheets. No automated alerts.

Impact: Risk of missing critical submission windows.

Desired: Automated deadline alerts at 30/14/7/3/1 days out.

4. No Path-Specific Dashboards

Problem: Single pipeline view for all paths.

Impact: Can't see health of each acquisition strategy.

Desired: Dashboard per path showing count, value, weighted value, upcoming deadlines.

5. Contact/Relationship Tracking Gaps

Problem: Government contacts, prime contractors, and teaming partners mixed together.

Impact: Hard to track relationship strength for Path 3 and Path 4.

Desired: Contact segmentation by type with relationship scoring.

6. No SAM.gov Integration Workflow

Problem: Manually entering opportunities found on SAM.gov.

Impact: Double data entry, potential for errors.

Desired: Streamlined workflow with consistent fields.
